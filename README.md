# mate_mini
Quick HowTo:
- post initial boot of ubuntu mini.iso
- sudo apt-get install git

- git clone https://github.com/midfingr/mate_mini
- cd mate_mini
- chmod +x ins
- type: ./ins ## supply password
- reboot
- proceed to the optional post_ins:
- in a terminal:
- cd mate_mini
- chmod +x post_ins
- type: ./post_ins ## supply password
- reboot
Have Fun! :)
